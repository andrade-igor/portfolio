# Igor Andrade — Portfólio

Site estático com portfólio profissional e currículo imprimível.

- **Portfólio**: [https://andrade-igor.github.io/portfolio/](https://andrade-igor.github.io/portfolio/)
- **Currículo (CV)**: [https://andrade-igor.github.io/portfolio/cv.html](https://andrade-igor.github.io/portfolio/cv.html)

## Estrutura

```
portfolio/
├── index.html       # Portfólio (home)
├── cv.html          # Currículo imprimível (A4)
├── styles.css       # Estilos (tema claro, Fraunces + Inter)
└── assets/
    ├── igor.png          # Foto (Amazon IA Summit 2025)
    └── qr-portfolio.png  # QR code do CV apontando pro portfólio
```

## Desenvolver localmente

```bash
open index.html
```

## Imprimir o CV

```bash
open cv.html
# → Cmd+P → Salvar como PDF (A4)
```
